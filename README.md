# Si t'es U — Site officiel de l'association

Site web officiel de l'association **Si t'es U**, association étudiante des cités universitaires du CLOUS de Brest.

---

## À propos

**Si t'es U** est une association créée le **15 octobre 2015**, dont le nom fait référence à la « Cité U » (cité universitaire). Elle a pour objectifs de :

- Représenter et défendre les intérêts des étudiants résidents en cité universitaire du CLOUS de Brest  
- Animer les cités par des manifestations diverses  
- Lutter contre l'isolement social des étudiants  
- Faire le lien entre les résidents et les instances du CROUS de Rennes-Bretagne  

Le siège de l'association se trouve à la cité universitaire de **Kergoat** (4 rue des Archives, 29200 Brest).

## Utilisation

1. Cloner ou télécharger le dépôt.  
2. Ouvrir `index.html` dans un navigateur (double-clic ou via un serveur local).  
3. Naviguer via le menu : Présentation, Événements, Membres, Collaborations.

Aucune installation ni dépendance externe n’est requise.
